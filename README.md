🎯 New Year New Mission XI – Landing Page

New Year New Mission XI is a modern and responsive landing page built using HTML, CSS, and JavaScript. The project focuses on goal-oriented UI design with a clean layout and smooth responsiveness.

🚀 Live Demo

🔗 https://new-year-new-mission-xi.vercel.app/

📌 Features
🎯 Goal/mission-based landing page UI
📱 Fully responsive design (mobile, tablet, desktop)
🧭 Clean section-based layout
🎨 Modern and minimal design style
⚡ Lightweight and fast-loading frontend
🧩 Organized structure for scalability
